Je project is naar GitHub verzonden.

### Hoe los ik een 'Failed' build op?
1. Ga naar het tabblad **Actions** op GitHub.
2. Klik op de gefaalde build (rood kruisje).
3. Klik op **"Build Android APK"** en kijk bij de stap **"Run APK Forge Script"**.
4. Meestal is de fout een typefout in de URL of een ongeldige App ID.

### Tips voor succes:
- Gebruik een geldige App ID zoals `com.mijnbedrijf.app`.
- Zorg dat je Website URL begint met `https://`.
- Wacht geduldig (5-7 minuten) tot de build klaar is.
